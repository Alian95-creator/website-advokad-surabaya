# Law Office Website – Purwo Widodo SH MH & Partner

Website resmi untuk **Kantor Advokat Purwo Widodo SH MH & Partner** yang berlokasi di Surabaya.  
Website ini dibuat sebagai website profil untuk menampilkan layanan hukum, testimoni klien, dan informasi kantor.

Live Website:  
https://alian95-creator.github.io/website-advokad-surabaya/

---

## Project Overview

Website ini dibuat menggunakan **HTML, CSS, dan JavaScript (Vanilla JS)** tanpa framework.  
Fokus utama dari project ini adalah membuat website yang:

- Responsive
- SEO Friendly
- User friendly
- Lightweight

Website ini menampilkan beberapa fitur utama seperti layanan hukum, testimoni klien, dan informasi kantor yang terintegrasi dengan Google Maps.

---

## Features

- Responsive navigation bar (mobile hamburger menu)
- Background hero slider
- Layanan hukum section dengan interactive card
- Clickable layanan detail (accordion behavior)
- Testimoni klien slider
- Google Maps integration
- WhatsApp floating consultation button
- SEO meta tags optimization
- Structured data (Schema.org)

---

## Tech Stack

Project ini dibuat menggunakan teknologi berikut:

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome Icons
- Google Maps Embed API

Tidak menggunakan framework seperti React atau Vue agar website tetap **ringan dan cepat di-load**.

## Key Implementation

Beberapa implementasi utama dalam project ini:

### Background Image Slider
Hero section menggunakan JavaScript untuk mengganti background image secara otomatis setiap beberapa detik.

### Interactive Layanan Section
Setiap card layanan dapat diklik untuk menampilkan detail layanan menggunakan **JavaScript toggle interaction**.

### Testimoni Slider
Testimoni klien ditampilkan dalam bentuk horizontal scroll slider yang dapat digeser pada perangkat mobile maupun desktop.

### WhatsApp Consultation Button
Tombol floating WhatsApp memungkinkan pengguna langsung menghubungi kantor hukum untuk konsultasi.

---

## SEO Optimization

Website ini dioptimasi untuk SEO dengan menggunakan:

- Meta description
- Meta keywords
- Open Graph tags
- Canonical URL
- Schema.org structured data

Tujuannya agar website lebih mudah ditemukan di Google untuk pencarian seperti:

- Pengacara Surabaya
- Advokat Surabaya
- Konsultasi hukum Surabaya
